题目 1：交替打印 abc

题目描述

使用 Go 实现 3 个并发执行的 goroutine，分别负责打印字符 `a`、`b`、`c`。

要求它们严格按顺序循环输出：

`abcabcabc...`

例如，当循环次数为 `n = 5` 时，最终输出应为：

`abcabcabcabcabc`

具体要求

1. 必须使用 **3 个 goroutine**（每个 goroutine 只负责一个字符）。

2. 输出顺序必须严格正确，不能出现乱序（如 `acb`、`aab` 等）。

3. 总输出次数由参数 `n` 控制，最终长度应为 `3 * n`。

4. 程序应能正常退出，不能出现 goroutine 泄漏或死锁。

5. 建议将逻辑封装为函数，例如：


func PrintABC(n int)

输入输出说明

  • 输入：一个正整数 `n`，表示 `abc` 的重复次数。
  • 输出：按顺序打印的字符串序列（可打印到标准输出）。

示例

示例 1

  • 输入：`n = 1`
  • 输出：`abc`

示例 2

  • 输入：`n = 3`
  • 输出：`abcabcabc`

示例 3

  • 输入：`n = 0`
  • 输出：空（不打印）

考察点

1. goroutine 的协作与调度控制。

2. channel / sync 等并发同步机制的正确使用。

3. 并发程序的结束条件设计（避免死锁和资源泄漏）。

4. 代码可读性与可扩展性（如扩展到打印 `abcd...`）。

补充说明（可选约束）

  • 可要求候选人至少给出一种基于 `channel` 的实现。
  • 可追问是否能在不使用 `time.Sleep` 的前提下稳定输出。
  • 可追问如何支持任意字符集合（如 `[]string{"A", "B", "C", "D"}`）的有序循环打印。

代码实现

对应题解代码与测试文件按目录展示，便于对照阅读与回顾。

1_print_abc.go

src/1_print_abc.go 
package src

import (
	"fmt"
	"sync"
)

func PrintABC(n int) {
	ch1 := make(chan struct{}, 1)
	ch2 := make(chan struct{}, 1)
	ch3 := make(chan struct{}, 1)
	var wg sync.WaitGroup

	ch1 <- struct{}{}
	wg.Add(1)
	go func() {
		defer wg.Done()
		for i := 0; i < n; i++ {
			<-ch1
			fmt.Println("a")
			ch2 <- struct{}{}
		}
	}()

	wg.Add(1)
	go func() {
		defer wg.Done()
		for i := 0; i < n; i++ {
			<-ch2
			fmt.Println("b")
			ch3 <- struct{}{}
		}
	}()

	wg.Add(1)
	go func() {
		defer wg.Done()
		for i := 0; i < n; i++ {
			<-ch3
			fmt.Println("c")
			ch1 <- struct{}{}
		}
	}()

	wg.Wait()
}

package test

import (
	"bytes"
	"io"
	"os"
	"strings"
	"testing"

	"go_coding_noalgorithm/src"
)

func captureStdout(t *testing.T, fn func()) string {
	t.Helper()

	oldStdout := os.Stdout
	r, w, err := os.Pipe()
	if err != nil {
		t.Fatalf("failed to create pipe: %v", err)
	}

	os.Stdout = w
	fn()

	_ = w.Close()
	os.Stdout = oldStdout

	var buf bytes.Buffer
	if _, err := io.Copy(&buf, r); err != nil {
		t.Fatalf("failed to read stdout: %v", err)
	}
	_ = r.Close()

	return buf.String()
}

func TestPrintABC_OrderAndCount(t *testing.T) {
	got := captureStdout(t, func() {
		src.PrintABC(3)
	})

	want := "a\nb\nc\na\nb\nc\na\nb\nc\n"
	if got != want {
		t.Fatalf("unexpected output\nwant:\n%q\ngot:\n%q", want, got)
	}
}

func TestPrintABC_Zero(t *testing.T) {
	got := captureStdout(t, func() {
		src.PrintABC(0)
	})

	if strings.TrimSpace(got) != "" {
		t.Fatalf("expected empty output when n=0, got %q", got)
	}
}
