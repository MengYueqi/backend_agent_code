# 题目 3：至多包含两种不同数字的最长连续子序列

## 题目描述
给定一个整数序列 `nums`，请返回其中满足“**连续子序列中最多只包含 2 种不同数字**”的最长长度。  
你只需要返回长度，不需要返回具体子序列。

## 具体要求
1. 子序列必须是连续的（即数组中的一个连续区间）。
2. 统计区间内不同数字的个数，要求不超过 2。
3. 返回满足条件的最长区间长度（`int`）。
4. 当输入为空序列时，返回 `0`。
5. 允许数字重复，也允许出现负数。

## 输入输出说明
- 输入：整数切片 `nums []int`。
- 输出：整数 `int`，表示满足条件的最长连续子序列长度。

建议函数签名：

```go
func MaxLenAtMostTwoDistinct(nums []int) int
```

## 示例
### 示例 1（正常场景）
- 输入：`nums = [1,2,3,2,2]`
- 输出：`4`
- 说明：最长区间是 `[2,3,2,2]`，包含数字 `2` 和 `3` 两种。

### 示例 2（正常场景）
- 输入：`nums = [1,2,1,1,3]`
- 输出：`4`
- 说明：最长区间是 `[1,2,1,1]`，只包含 `1` 和 `2` 两种。

### 示例 3（边界场景）
- 输入：`nums = []`
- 输出：`0`
- 说明：空输入没有可选区间。
